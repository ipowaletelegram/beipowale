🚀 JAIMIN PARMAR's Dashboard Features

Powerful tools for modern users

Core Features

  • OTP Login: Email-only login with secure OTP verification.
  • QR Code Generator: Create QR codes instantly and download them.
  • User Tracking: Know when and where a user last logged in.
  • Google Sheets Logging: Activity logs are sent to a linked spreadsheet.
  • Dark Mode: Toggle-friendly, elegant dark UI.
  • Responsive Design: Works on desktop, tablet, and mobile.
  • Location Tracking: Allows users to share their real-time GPS data.
  • Live Clock: Shows time and date on dashboard.
  • Password Manager: Integrated password vault (hosted separately).
  • Admin Dashboard: View user stats, messages, and tools from a unified view.
